Privacy Policy

Last updated: May 2026 · Applies to CutSmith Subtitle Studio (macOS desktop app)

CutSmith Subtitle Studio is a local desktop application. This policy explains what data is collected, how it is used, and your rights.

1. Your footage and audio stay on your machine

CutSmith Subtitle Studio does not upload your video or audio files to any server. All transcription and subtitle processing runs locally on your Mac using on-device machine learning models (Whisper via Apple MLX). Your media never leaves your drive during normal operation.

2. What the app accesses

  • Video and audio files — only files you explicitly open or drop into the app. Read-only access for transcription; no copies are made outside the output directory you choose.
  • Output directory — the folder you select for SRT/TXT export. The app writes only to locations you designate.
  • Model files — Whisper model weights downloaded to your local machine on first use. Downloaded once over HTTPS; not updated automatically.

3. Email collection (optional)

The CutSmith website offers an optional email sign-up to receive notifications about new alpha builds. This is entirely voluntary. If you submit your email:

  • It is stored for the sole purpose of sending release notifications.
  • It is not shared with third parties or used for advertising.
  • You can request deletion at any time by emailing hello@cutsmith.io.

If you do not submit your email, no personal data is collected by the website beyond standard server access logs (IP address, browser, referring URL), which are retained for up to 30 days for security purposes.

4. Analytics

The CutSmith website does not use third-party analytics scripts (no Google Analytics, no tracking pixels). No cookies are set for tracking purposes.

5. The desktop app

The desktop app does not connect to the internet during normal operation. No usage data, crash reports, or telemetry are sent from the app. Error information stays on your machine.

The app checks for model file availability on first launch. This is a one-time download request to a static file host. No identifying information is sent beyond a standard HTTPS request.

6. Third-party services

CutSmith uses GitHub for source code hosting and release distribution. Downloads from GitHub are subject to GitHub's privacy policy.

7. Children

CutSmith is a professional tool intended for adult editors and post-production teams. We do not knowingly collect data from anyone under 16.

8. Changes to this policy

If this policy changes materially, the updated version will be posted here with a revised date. For significant changes affecting email subscribers, notification will be sent to the address on file.

9. Contact

Questions about this policy or data deletion requests: hello@cutsmith.io
